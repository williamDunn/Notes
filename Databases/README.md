## What is a Database?

-  A structured set of data held in a computer

----------------------
#### SQL vs NoSQL Databases

-  SQL Databases are relational **|** NoSQL Databases are non-relational

#### Why use SQL vs NoSQL DBs

-  SQL = Organized
-  NoSQL = Easier to store different types of data
