# Wrapper
------------------
Wrapper classes provide a way to use primitive data types (int, boolean, etc..) as objects.

| Primitive Data Type      | Wrapper Class |
| ----------- | ----------- |
| byte      | Byte       |
| short   | Short        |
| int      | Integer       |
| long   | Long        |
| float      | Float       |
| double   | Double        |
| boolean      | Boolean       |
| char   | Character        |
